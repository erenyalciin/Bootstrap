Tanım Listesi
Tanım listesi, terimlerin açıklamalarıyla birlikte listelendiği listeleme türüdür. <dt> etiketi terimleri, <dd> etiketi ise açıklamaları tanımlar.
HTML
HTML, web sayfaları ve uygulamaları oluşturmak için kullanılan işaretleme dilidir.
CSS
CSS, bir işaretleme diliyle yaratılmış belgenin sunumunu tanımlamak için kullanılan bir stil sayfası dilidir.
JavaScript
JavaScript, web sayfaları ve web tabanlı uygulamalarda etkileşimli yapılar oluşturmamızı sağlayan bir betik dilidir.
